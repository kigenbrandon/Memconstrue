# Memconstrue
The way to learn about memory management and way through to build scripts to learn security from scratch.

In this repo I'm going  to build an OS all the way to creating applications on top of it and having it run and test for security issues to get a deep dive into the world application security. Buckle up for a long journey and new discoveries with minimal use of AI because learning is getting your hands dirty and building a reputation towards automation and insights of security.

In each module of the project I will write separate README files for guidance have fun learning with me ciao.
